
Input/Output makes a program more dynamic. Users can interact directly with the program, and the output changes depending on the input. By saving data to a file, the data can persist after the program terminates.

## Scanner

The `Scanner` class is a useful tool in Java that can read data from various input sources, such as the keyboard or files. `Scanner` reads input data sequentially and converts it into the required data type, such as an integer, floating-point number, or string.

A `Scanner` can be used simply by creating a `Scanner` object and passing `System.in` as an argument. `System.in` is the standard input stream. When it is passed to a `Scanner`, it creates a `Scanner` object that can read keyboard input.

```java
public static void main(String[] args) {  
    Scanner sc = new Scanner(System.in);  
}
```
