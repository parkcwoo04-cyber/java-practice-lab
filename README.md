# Practice System

The exercises in this repository are generated through a structured AI-assisted learning process.

Problems are designed to:
- match my current learning level
- avoid concepts that have not yet been studied
- focus on practical Java fundamentals
- remain solvable within 20–30 minutes
- reinforce weak concepts through targeted repetition

Each exercise follows a structured format including:
- Class Name
- Difficulty
- Estimated Time
- Topics
- Problem Description
- Requirements
- Example Output
- Learning Objectives

Difficulty is based on overall problem complexity rather than only on Java concepts.

---

# Reflection-Based Learning

A major focus of this repository is reflection-driven improvement.

After completing each exercise, I document:
- my initial approach
- mistakes and debugging process
- conceptual misunderstandings
- improvements made after review
- key takeaways from the exercise

This process helps reinforce concepts and identify recurring weaknesses over time.

The objective is to transform mistakes into reusable learning assets rather than temporary fixes.

Example reflection topics include:
- equals() vs ==
- substring() index confusion
- loop boundary mistakes
- conditional logic errors

---

# Weakness Tracking

This learning system tracks recurring mistakes and weak areas across exercises.

Rather than immediately moving to harder topics, reinforcement exercises are generated to strengthen:
- logical thinking
- syntax accuracy

- code readability
- problem decomposition

The objective is to build consistent understanding through repetition, reflection, and targeted reinforcement.
